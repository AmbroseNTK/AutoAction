# AutoAction
Tool xây dựng kịch bản cho PC tự làm việc
----
Hướng dẫn
B1. Tải bản released mới nhất
B2. Cài Python 2.x hoặc 3.x
B3. Chạy file install.bat (Máy phải có công cụ pip của python)
B4. Sử dụng AutoAction.exe
